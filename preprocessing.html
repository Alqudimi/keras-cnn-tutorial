الرئيسية معالجة الصور

معالجة وإعداد مجموعات الصور للتعلم العميق

يعد إعداد البيانات خطوة حاسمة في بناء نماذج التعلم العميق الفعالة. هذا الدليل يشرح التقنيات الأساسية لضمان جودة وتنوع مجموعة الصور باستخدام TensorFlow و Keras.

المقدمة: أهمية إعداد البيانات

"القمامة في الداخل، القمامة في الخارج" - هذا المبدأ ينطبق تمامًا على التعلم العميق. مهما كانت بنية النموذج متطورة، فإن جودة النتائج تعتمد بشكل كبير على جودة البيانات المستخدمة في التدريب. إعداد البيانات الجيد يمكن أن يحسن أداء النموذج بشكل كبير ويقلل من الحاجة إلى بنى أكثر تعقيدًا.

إزالة التشويش وتحسين جودة الصور

الصور المشوشة أو غير الواضحة يمكن أن تضلل النموذج وتؤدي إلى ضعف في التعميم. هناك طريقتان رئيسيتان للتعامل مع التشويش: المعالجة التقليدية باستخدام الفلاتر، أو استخدام نماذج التعلم العميق.

الفلاتر التقليدية في OpenCV

التقنية الوصف متى تستخدم؟
Gaussian Blur تنعيم الصورة وتقليل التشويش العشوائي لإزالة التشويش الخفيف والضوضاء عالية التردد
Median Filter فعال في إزالة ضوضاء "الملح والفلفل" عندما تكون الضوضاء نقاط ساطعة أو مظلمة معزولة
Bilateral Filter يحافظ على حواف الصورة أثناء إزالة التشويش عندما يكون الحفاظ على الحواف أمرًا بالغ الأهمية
Non-Local Means تقنية متقدمة تقارن بين أجزاء مختلفة من الصورة للحصول على أفضل النتائج مع الحفاظ على التفاصيل
Python 
import cv2
import numpy as np

# تحميل الصورة المشوشة
noisy_image = cv2.imread('noisy_image.jpg')

# ═══════════════════════════════════════════════════
# 1. فلتر الوسيط (Median Filter)
# مثالي لضوضاء الملح والفلفل
# ═══════════════════════════════════════════════════
median_denoised = cv2.medianBlur(noisy_image, 5)  # حجم النواة 5×5

# ═══════════════════════════════════════════════════
# 2. فلتر ثنائي (Bilateral Filter)
# يحافظ على الحواف
# ═══════════════════════════════════════════════════
bilateral_denoised = cv2.bilateralFilter(
    noisy_image, 
    d=9,           # قطر المنطقة
    sigmaColor=75, # قوة الفلترة للألوان المتشابهة
    sigmaSpace=75  # قوة الفلترة للمواقع المتقاربة
)

# ═══════════════════════════════════════════════════
# 3. إزالة التشويش غير المحلي (Non-Local Means)
# الأفضل للحفاظ على التفاصيل
# ═══════════════════════════════════════════════════
# تحويل إلى تدرج رمادي أولاً
gray_noisy = cv2.cvtColor(noisy_image, cv2.COLOR_BGR2GRAY)
nlm_denoised = cv2.fastNlMeansDenoising(
    gray_noisy, 
    None,
    h=30,              # قوة الفلترة
    templateWindowSize=7,
    searchWindowSize=21
)

# للصور الملونة
nlm_colored = cv2.fastNlMeansDenoisingColored(
    noisy_image,
    None,
    h=10,
    hColor=10,
    templateWindowSize=7,
    searchWindowSize=21
)

# حفظ النتائج
cv2.imwrite('median_denoised.jpg', median_denoised)
cv2.imwrite('bilateral_denoised.jpg', bilateral_denoised)
cv2.imwrite('nlm_denoised.jpg', nlm_colored)

print("✅ تم إزالة التشويش بنجاح!")

نصيحة: التعلم العميق لإزالة التشويش

يمكن استخدام نماذج مثل Autoencoders أو DnCNN لإزالة التشويش بشكل أكثر فعالية، خاصة إذا كانت طبيعة التشويش فريدة لمجموعة بياناتك. تُدرب على أزواج من الصور (مشوشة كمدخل، نظيفة كمخرج).

زيادة تنوع وحجم مجموعة الصور (Data Augmentation)

زيادة البيانات هي تقنية أساسية لمكافحة الإفراط في الملاءمة (Overfitting) وزيادة قوة تعميم النموذج. يتم ذلك عن طريق إنشاء نسخ معدلة من الصور الموجودة بتطبيق تحويلات متنوعة.

تقنيات التحويل الأساسية

التحويل الوصف المعامل في Keras القيمة المقترحة
التدوير تدوير الصورة بزاوية عشوائية rotation_range 20-40 درجة
القلب الأفقي عكس الصورة أفقيًا horizontal_flip True
القلب العمودي عكس الصورة رأسيًا vertical_flip حسب طبيعة البيانات
التكبير/التصغير تكبير أو تصغير جزء من الصورة zoom_range 0.1-0.3
الإزاحة الأفقية تحريك الصورة يمينًا أو يسارًا width_shift_range 0.1-0.2
الإزاحة العمودية تحريك الصورة أعلى أو أسفل height_shift_range 0.1-0.2
القص (Shear) إمالة الصورة على محور shear_range 0.1-0.2
تغيير السطوع تعديل سطوع الصورة brightness_range [0.8, 1.2]

استخدام ImageDataGenerator

Python 
from tensorflow.keras.preprocessing.image import ImageDataGenerator

# ═══════════════════════════════════════════════════
# إعداد مولد البيانات مع تقنيات الزيادة المتنوعة
# ═══════════════════════════════════════════════════
train_datagen = ImageDataGenerator(
    # التطبيع (معالجة أولية)
    rescale=1./255,
    
    # تحويلات هندسية
    rotation_range=40,          # تدوير حتى 40 درجة
    width_shift_range=0.2,      # إزاحة أفقية 20%
    height_shift_range=0.2,     # إزاحة عمودية 20%
    shear_range=0.2,            # قص 20%
    zoom_range=0.2,             # تكبير/تصغير 20%
    
    # قلب الصور
    horizontal_flip=True,       # قلب أفقي
    vertical_flip=False,        # عادة لا نحتاجه للصور الطبيعية
    
    # كيفية ملء البكسلات الجديدة
    fill_mode='nearest'         # استخدام أقرب بكسل
)

# مولد للتحقق (بدون زيادة بيانات)
val_datagen = ImageDataGenerator(rescale=1./255)

# ═══════════════════════════════════════════════════
# تحميل البيانات من المجلدات
# ═══════════════════════════════════════════════════
train_generator = train_datagen.flow_from_directory(
    'data/train',
    target_size=(224, 224),     # توحيد الحجم
    batch_size=32,
    class_mode='categorical',   # تصنيف متعدد الفئات
    shuffle=True                # خلط البيانات
)

val_generator = val_datagen.flow_from_directory(
    'data/validation',
    target_size=(224, 224),
    batch_size=32,
    class_mode='categorical',
    shuffle=False               # لا نخلط بيانات التحقق
)

print(f"عدد الفئات: {train_generator.num_classes}")
print(f"أسماء الفئات: {train_generator.class_indices}")

استخدام طبقات Keras الحديثة

Python 
import tensorflow as tf
from tensorflow.keras import layers

# ═══════════════════════════════════════════════════
# طبقات زيادة البيانات الحديثة (TensorFlow 2.x)
# تُطبق كجزء من النموذج نفسه
# ═══════════════════════════════════════════════════
data_augmentation = tf.keras.Sequential([
    # القلب الأفقي
    layers.RandomFlip("horizontal"),
    
    # التدوير (0.1 = 10% من 360 درجة = 36 درجة)
    layers.RandomRotation(0.1),
    
    # التكبير/التصغير
    layers.RandomZoom(0.1),
    
    # تغيير السطوع
    layers.RandomBrightness(0.2),
    
    # تغيير التباين
    layers.RandomContrast(0.2),
])

# ═══════════════════════════════════════════════════
# تطبيق الزيادة على مجموعة البيانات
# ═══════════════════════════════════════════════════
def prepare_dataset(ds, augment=False):
    """إعداد مجموعة البيانات مع الزيادة الاختيارية"""
    # التطبيع
    normalization = layers.Rescaling(1./255)
    ds = ds.map(lambda x, y: (normalization(x), y))
    
    # تطبيق الزيادة على بيانات التدريب فقط
    if augment:
        ds = ds.map(
            lambda x, y: (data_augmentation(x, training=True), y)
        )
    
    return ds

# استخدام مع tf.data.Dataset
train_ds = prepare_dataset(train_ds, augment=True)
val_ds = prepare_dataset(val_ds, augment=False)

تحذير: زيادة البيانات على التدريب فقط!

لا تطبق زيادة البيانات على مجموعات التحقق أو الاختبار. هذه المجموعات يجب أن تمثل البيانات الحقيقية التي سيواجهها النموذج في الإنتاج. الزيادة تُطبق فقط أثناء التدريب.

تقسيم وإعداد مجموعة البيانات

يجب تقسيم مجموعة البيانات إلى ثلاثة أقسام لتقييم أداء النموذج بشكل صحيح ومنع تسرب البيانات (Data Leakage).

القسم الغرض النسبة الموصى بها ملاحظات
التدريب (Training) تدريب النموذج وتعلم الأنماط 70% - 80% يُستخدم لتحديث أوزان النموذج
التحقق (Validation) ضبط المعاملات ومراقبة Overfitting 10% - 15% يُستخدم أثناء التدريب لكن لا يُدرب عليه
الاختبار (Test) تقييم الأداء النهائي 10% - 15% يُستخدم مرة واحدة فقط في النهاية
Python 
import tensorflow as tf

التطبيع (Normalization)

التطبيع هو أهم خطوة في المعالجة المسبقة. يتم تحويل قيم البكسل (0-255) إلى نطاق أصغر (0-1) مما يساعد في تسريع التقارب ومنع المشاكل العددية.

لماذا التطبيع مهم؟

  • تسريع التقارب: قيم أصغر تعني تدرجات أكثر استقرارًا
  • تحسين الاستقرار العددي: يمنع overflow وunderflow
  • أداء أفضل: معظم الخوارزميات تعمل بشكل أفضل مع بيانات مطبعة
  • توافق: النماذج المُدربة مسبقًا تتوقع بيانات مطبعة

تحسين الأداء (Performance Optimization)

لتحسين سرعة التدريب، يجب تطبيق تقنيات جلب البيانات الفعالة. هذا يمنع وحدة المعالجة المركزية من أن تصبح عنق الزجاجة بينما تنتظر وحدة GPU للبيانات.

تقنيات التحسين

التقنية الوصف الفائدة
.cache() يحفظ الصور في الذاكرة بعد التحميل الأول يقلل زمن قراءة القرص في الدورات اللاحقة
.shuffle() يخلط ترتيب الصور يمنع النموذج من تعلم ترتيب معين
.prefetch() يجلب الدفعات التالية بالتوازي مع التدريب يقلل زمن الانتظار
Python 
import tensorflow as tf

# ═══════════════════════════════════════════════════
# تحسين أداء خط أنابيب البيانات
# ═══════════════════════════════════════════════════
AUTOTUNE = tf.data.AUTOTUNE

# مجموعة التدريب
train_ds = train_ds.cache()           # حفظ في الذاكرة
train_ds = train_ds.shuffle(1000)     # خلط البيانات
train_ds = train_ds.prefetch(         # جلب مسبق
    buffer_size=AUTOTUNE              # TensorFlow يحدد الحجم الأمثل
)

# مجموعة التحقق (بدون خلط)
val_ds = val_ds.cache()
val_ds = val_ds.prefetch(buffer_size=AUTOTUNE)

# ═══════════════════════════════════════════════════
# للمجموعات الكبيرة جدًا: استخدام cache على القرص

مثال كامل: إعداد مجموعة البيانات

Python 
import tensorflow as tf
from tensorflow.keras import layers
import os

# ═══════════════════════════════════════════════════
# المعلمات الأساسية
# ═══════════════════════════════════════════════════
DATA_DIR = 'data/train'
BATCH_SIZE = 32
IMG_HEIGHT = 180
IMG_WIDTH = 180
AUTOTUNE = tf.data.AUTOTUNE

# ═══════════════════════════════════════════════════
# 1. تحميل وتقسيم البيانات
# ═══════════════════════════════════════════════════
print("--- تحميل وتقسيم البيانات ---")

train_ds = tf.keras.utils.image_dataset_from_directory(
    DATA_DIR,
    validation_split=0.2,
    subset="training",
    seed=123,
    image_size=(IMG_HEIGHT, IMG_WIDTH),
    batch_size=BATCH_SIZE
)

val_ds = tf.keras.utils.image_dataset_from_directory(
    DATA_DIR,
    validation_split=0.2,
    subset="validation",
    seed=123,
    image_size=(IMG_HEIGHT, IMG_WIDTH),
    batch_size=BATCH_SIZE
)

class_names = train_ds.class_names
print(f"الفئات المكتشفة: {class_names}")

# ═══════════════════════════════════════════════════
# 2. التطبيع
# ═══════════════════════════════════════════════════
print("--- التطبيع ---")
normalization_layer = layers.Rescaling(1./255)

train_ds = train_ds.map(lambda x, y: (normalization_layer(x), y))
val_ds = val_ds.map(lambda x, y: (normalization_layer(x), y))

# ═══════════════════════════════════════════════════
# 3. زيادة البيانات (للتدريب فقط)
# ═══════════════════════════════════════════════════
print("--- زيادة البيانات ---")
data_augmentation = tf.keras.Sequential([
    layers.RandomFlip("horizontal"),
    layers.RandomRotation(0.1),
    layers.RandomZoom(0.1),
])

train_ds = train_ds.map(
    lambda x, y: (data_augmentation(x, training=True), y)
)

# ═══════════════════════════════════════════════════
# 4. تحسين الأداء
# ═══════════════════════════════════════════════════
print("--- تحسين الأداء ---")
train_ds = train_ds.cache().shuffle(1000).prefetch(buffer_size=AUTOTUNE)
val_ds = val_ds.cache().prefetch(buffer_size=AUTOTUNE)

# ═══════════════════════════════════════════════════
# 5. التحقق من شكل البيانات
# ═══════════════════════════════════════════════════
print("--- شكل البيانات ---")
for image_batch, labels_batch in train_ds.take(1):
    print(f"شكل دفعة الصور: {image_batch.shape}")
    print(f"شكل دفعة التسميات: {labels_batch.shape}")
    print(f"نطاق قيم البكسل: {image_batch.numpy().min():.3f} - {image_batch.numpy().max():.3f}")

print("\n✅ تم إعداد مجموعات البيانات وهي جاهزة للتدريب!")
